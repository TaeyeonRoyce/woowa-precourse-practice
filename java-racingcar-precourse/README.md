# 🚗 자동차 경주 게임

### ⛳️ 프로젝트 소개

`목적` : 가장 멀리 나가는 자동차 선별하기

`입력` : 자동차의 종류, 진행 횟수



👨‍💻 `예시`

```
경주할 자동차 이름을 입력하세요.(이름은 쉼표(,) 기준으로 구분)
pobi,woni,jun
시도할 회수는 몇회인가요?
5

실행 결과
pobi : -
woni : 
jun : -

pobi : --
woni : -
jun : --

pobi : ---
woni : --
jun : ---

pobi : ----
woni : ---
jun : ----

pobi : -----
woni : ----
jun : -----

최종 우승자: pobi, jun
```



`입력 예외사항` 

- `자동차` : 자동차의 크기는 5이하이며  `,`로 구분됨
- `횟수` : 0보다 큰 정수



### 구현

1. 프로그램 구조 및 객체 설정
2. 자동차 입력 및 예외처리
3. 회수 입력 및 예외처리
4. 실행 결과 출력
5. 최종 우승자 출력
